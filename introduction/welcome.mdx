Natively is an AI-powered platform for building, iterating on, and publishing full-stack web applications. Describe what you want in natural language, preview changes in real time, and publish to a live URL when you're ready.

## What you can do

- **Build with AI** — Chat with specialized agents to generate and refine React web apps
- **Preview instantly** — See your app running in a live preview as you build
- **Publish in one click** — Deploy to a shareable URL on `*.nativelyai.app`
- **Collaborate** — Work with teammates in shared workspaces with role-based access
- **Scale with plans** — Start free, upgrade as you need more credits, projects, and features

## How Natively is organized

| Concept | Description |
|---------|-------------|
| **Workspace** | Your team's home for projects, billing, and members |
| **Project** | A single app you're building — has its own chat, code, and preview |
| **Agent** | A specialized AI assistant (Builder, Product Architect, and more) |
| **Credits** | Usage units consumed when you chat with AI or run generation |
