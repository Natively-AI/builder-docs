## v1.0.6 — May 2026

### Mobile project experience

- New mobile header and bottom navigation for switching between chat, preview, tasks, and versions
- Improved mobile file explorer and responsive project view

### Billing improvements

- Promo codes auto-proceed to checkout when valid
- Plan upgrades wait for payment confirmation before applying
- Cancelled Stripe checkout returns to billing settings instead of a 404
- Removed project and member caps on paid Pro plans
- Stronger credit enforcement across all model flows

### Builder UX

- Contextual "tips for next step" suggestions in the chat composer
- Visual edit selections can be sent directly to chat
- Ask-question options render as full-width stacked buttons

### Domains

- Fixed regression when updating platform subdomains

### Admin (internal)

- Admin billing tooling for customer lookup, promo management, and credit operations

See the full release history in the [GitHub changelog directory](https://github.com/Natively-AI/conductor-2.0/tree/dev/changelog).

### Highlights from v1.0.0

- Full-stack AI app builder with live preview
- Project publishing to `*.nativelyai.app`
- Custom domain support with DNS verification
- Workspace collaboration with role-based access
- Credit-based billing with Stripe subscriptions
- Multiple specialized AI agents (Builder, Product Architect, Feedback, and more)
- Marketplace for sharing components
- Version history and revert
